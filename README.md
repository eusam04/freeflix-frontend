## ✨ Sobre o Projeto

O **Freeflix** é uma aplicação front-end inspirada na interface da Netflix, desenvolvida com o objetivo de simular uma plataforma de streaming.

O projeto foi criado durante uma imersão prática, com foco em:

- construção de interfaces modernas  
- organização de conteúdo visual  
- responsividade e experiência do usuário  
- boas práticas de desenvolvimento front-end  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🧠 Diferenciais

- Interface inspirada em plataformas reais de streaming  
- Organização visual baseada em catálogos de filmes  
- Utilização de ferramentas de IA como apoio no desenvolvimento  
- Foco em experiência do usuário
